User Manual for GitHub Data Explorer Website

Welcome to the GitHub Data Explorer! This manual will guide you through the steps to use the website effectively, whether you choose to provide your own GitHub token or not. This website will analyze data from GitHub including user repositories, organizations, and user accounts.

1. Getting Started

Access the Website: Open your web browser and navigate to the GitHub Data Explorer Website

2. Using the Website Without a GitHub Token

If you prefer not to provide a GitHub token, you can still analyze public information from GitHub.

  1. Choosing type of analysis:
    • On the homepage, you will see options in header include: repository, user, organization
    • Click on options you want to analyze.
  2. Provide GitHub information
    • Enter your name of Github account or Github repository.
    • Skip the tokens field.
  3. Click confirm button and wait for a minute while system load data.

3. Using the Website with a GitHub Token

Providing a GitHub token allows the website to access more detailed information about your GitHub account, including private repositories and higher rate limits for API requests. The website will use your token to fetch and display detailed information, including private repositories, detailed contribution stars, and more.

  1. Choosing type of analysis:
    • On the homepage, you will see options in header include: repository, user, organization
    • Click on options you want to analyze.
  2. Generate a GitHub Token:
    • Go to your GitHub account settings.
    • Navigate to "Developer settings" and then "Personal access tokens."
    • Click "Generate new token" and select the necessary scopes (such as repo for repository access and user for user data).
    • Copy the generated token.
    • Notice: Read this page to have more detailed guidelines: GitHub Token Guidelines
  3. Provide GitHub information
    • Enter your name of Github account or Github repository.
    • Skip the tokens field.
  4. Click confirm button and wait for a minute while system load data.

4. Privacy and Security Guarantee

We take your privacy and the security of your data very seriously. Here is our commitment to you:

  • Token Usage: Your GitHub token is used solely to fetch data from GitHub and provide you with detailed insights.
  • No Storage: We do not store your GitHub token on our servers. Once the data is fetched, the token is discarded.
  • Data Privacy: Any data fetched using your GitHub token is not stored or shared. It is used only for displaying insights during your current session.
  • Security Measures: We employ industry-standard security measures to protect your data during transmission.

We recommend you use fine-granted access token which help you configure access policies with more secure. Your trust is important to us, and we are committed to maintaining the confidentiality and security of your private data.
